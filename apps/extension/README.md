## VoiceCoPilot Extension (MVP)

### Установка
1) Открой `chrome://extensions`
2) Включи режим разработчика
3) Нажми "Загрузить распакованное"
4) Выбери папку `apps/extension`

### Как работает
- Web UI отправляет `window.postMessage` с `type: startCapture`.
- Расширение захватывает звук вкладки и шлёт чанки обратно в страницу.

### Ограничения
- Захват системного аудио в браузере ограничен, MVP работает с аудио вкладки.
